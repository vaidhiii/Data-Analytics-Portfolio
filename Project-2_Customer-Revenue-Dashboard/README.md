Customer & Revenue Insights Dashboard (Excel)

Project Overview
This dashboard analyzes customer behavior and revenue patterns for January 2024.

What the Dashboard Shows
- Revenue by City  
- Revenue by Product  
- Revenue Share by Customer Type  
- Daily Revenue Trend  

Features
- Interactive slicers (City & Customer Type)  
- Clean, professional dashboard layout  
- Fully connected visuals  

Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers
